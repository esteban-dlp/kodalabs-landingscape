# Koda Labs --- Landing Page

Official landing page for **Koda Labs**, a technology company focused on
building software, automation systems, AI-powered tools, and digital
platforms that simplify complex workflows.

------------------------------------------------------------------------

## About Koda Labs

**Koda Labs** is a technology-driven company focused on building
practical solutions through software, automation, and digital tools.

Our mission:

> Use technology to remove friction from people's workflows and turn
> complex processes into simple systems.

Koda Labs builds:

-   SaaS products
-   Automation systems
-   AI-powered tools
-   Custom applications
-   Business platforms
-   Digital productivity systems

The company works with both **businesses and individuals**, helping them
automate repetitive tasks, simplify operations, and scale with better
systems.

## Tech Stack

This landing page is built using modern web technologies:

-   **Next.js**
-   **React**
-   **TypeScript**
-   **i18n (multi-language support)**
-   **SEO metadata configuration**
-   **Responsive design**
-   **Modern component architecture**

The project is designed to be clean, modular, and easy to scale as the
company grows.

## Getting Started

Install dependencies:

    npm install

Run the development server:

    npm run dev

Open in browser:

    http://localhost:3000

------------------------------------------------------------------------

## Internationalization

The project includes built-in **i18n support**.

Languages currently supported:

-   English
-   Spanish

Translations are stored inside:

    /messages

Each language file contains structured translation keys for scalable
multilingual support.

------------------------------------------------------------------------

## SEO

The landing page includes SEO configuration in `layout.tsx`, including:

-   Title
-   Description
-   Canonical URL
-   Open Graph metadata
-   Twitter metadata
-   Icon configuration

This ensures the website is optimized for search engines and social
sharing.
